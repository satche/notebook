# Numeric types

## Integer

Integer, often abbreviated as `int`, are used as "simple" numbers.

## Float

[Float](https://en.wikipedia.org/wiki/Floating-point_arithmetic) values are used when we need *real* numbers, which means with negative and decimal values. For example, `3.141`, `-1.2345`, etc.
 
$$ (-1)^S * 1.M * 2^{(E-127)} $$
