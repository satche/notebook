# Functions

[Function](https://wikipedia.org/wiki/function_(mathematics)) is a mathematical object that takes an input and produces an output (number, vector, etc).

![Graphical representation of functions](http://urquhartmath.weebly.com/uploads/4/2/5/8/4258996/667788_orig.png)

## Notation

$$f: X^n \to Y$$
Function $f$ from [domain](https://wikipedia.org/wiki/domain_of_a_function) $X$ with $n$ variables to [codomain](https://wikipedia.org/wiki/range_(mathematics)) $Y$.
Note that there is multiple example of different types of domains and codomains. Find more [here](https://wikipedia.org/wiki/function_(mathematics)).

## Convex / concave

There is two methods two know if a function is [convex](https://wikipedia.org/wiki/convex_function):

- If we trace a line between two points from the function and that it's on or above the function (in the interval between those two points)
- If the 2nd [derivative](derivative.md)(on a given interval)
  $f(x)=x^2$ is convex as $f'(x)=2x$, $f''(x)=2 > 0$

A function $f(x)$ is [concave](https://wikipedia.org/wiki/concave_function) when its opposite $-f(x)$ is convex.
